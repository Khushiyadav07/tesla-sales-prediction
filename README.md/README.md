# Tesla Sales Prediction Pipeline

## Project Overview

This project builds an end-to-end Machine Learning pipeline on Tesla sales and production data.

### Features

* Data Cleaning
* Exploratory Data Analysis (EDA)
* Feature Engineering
* Regression Modeling
* Hyperparameter Tuning
* Time Series Forecasting

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn
* Prophet

## Workflow

### Data Preprocessing

* Handled missing values
* Removed duplicates
* Converted date columns

### Feature Engineering

* Quarter extraction
* Month cyclic encoding
* Lag features
* Rolling averages

### Machine Learning

* Random Forest Regressor
* GridSearchCV Hyperparameter Tuning

### Forecasting

* Prophet Time Series Forecasting

## Results

Successfully predicted Tesla deliveries and generated future delivery forecasts.
